Project Title: RentPay –  Car Rental Web Application

This project is a full-stack Car Rental Management System designed to digitalize and simplify the entire vehicle booking process. It provides a smooth, user-friendly experience for customers to browse, compare, and rent cars, while also offering a powerful admin panel for managing cars, users, and rental transactions.

The system includes secure authentication, real-time booking, image uploads, and a polished modern UI built with React and Tailwind. The backend uses Node.js, Express, and MongoDB, ensuring strong performance, scalability, and clean API architecture.



Key Features:
• Admin Analytics Dashboard – View total cars, rentals, and key business
 insights.
• Advanced Car Filtering – Filter cars by category, brand, model, fuel type.
• Rental History Tracking – Users can view past and upcoming bookings in a 
 clean dashboard.
• Protected API Routes – Only authenticated users and admins can perform 
 sensitive operations.
• Real-Time Availability Check – Prevents double-booking by dynamically 
 verifying availability.
• Profile Management – Users can update personal details and manage 
 accounts easily.
• Fully Responsive UI – Works smoothly across mobile, tablet, and desktop 
 screens.
• Optimized API Endpoints – Clean and modular Express controller–route 
 architecture.
• MongoDB Schema Design – Well-structured models for users, cars, and rental 
 records.
• Performance Optimized Frontend – Built using Vite for fast loading and 
 smooth interactions.


Technologies & Frameworks Used:

• Frontend: React.js, Tailwind CSS, Vite, Axios, React Router
• Backend: Node.js, Express.js, MongoDB, Mongoose
• Authentication: JWT & bcrypt
• File Handling: Multer & ImageKit integration
• Environment Management: dotenv
